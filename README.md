# MIF39

Pour lancer l'application :
* Compiler le serveur dans BASE_C++
<code>cd BASE_C++</code>
<code>mkdir build</code>
<code>cd build</code>
<code>cmake ..</code>
<code>make</code>
* Lancer le serveur se trouvant dans <code>build/bin/</code> avec la commande <code>./ServerInTheMiddle _Monde.bin_</code>
* Lancer le client Unity en chargeant la scène <code>Unity/Assets/testLoad.unity</code>
* Modifier l'adresse IP du serveur (variable associée à la caméra)
* Lancer la scène !
